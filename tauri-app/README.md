# Kaotik Tauri UI

Bu klasor, `kaotik` Rust motoru icin capraz platform Tauri arayuzunu icerir.

## Gereksinimler

1. Node.js 18+
2. Rust stable
3. Tauri sistem bagimliliklari

## Gelistirme

```bash
cd tauri-app
npm install
npm run tauri dev
```

## Uretim Build

```bash
cd tauri-app
npm install
npm run tauri build
```

## Desteklenen Akislar

1. Encrypt: kaotik, aes, kyber
2. Decrypt: kaotik, aes, kyber
3. Verify
4. Password validation

Kyber modu icin key dosya yolu zorunludur.
